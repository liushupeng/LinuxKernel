- [GCC 在线文档](https://gcc.gnu.org/onlinedocs/)（包括 GCC、CPP 等）

- [GNU 在线文档](https://www.gnu.org/manual/manual.html)（除了上述两个，还包括 Make 等）
- [Linux Kernel 在线文档](https://www.kernel.org/doc/html/latest/)
